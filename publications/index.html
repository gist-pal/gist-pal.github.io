International Papers

In the fields of Software Security, Software Engineering, and Programming Languages, most influential papers are published at top-tier international conferences rather than international journals. The following conferences are typically considered as top-tiers.

  • Software Security: IEEE S&P, USENIX Security, ACM CCS, NDSS
  • Software Engineering: ICSE, FSE, ASE
  • Programming Languages: PLDI, POPL, CAV, OOPSLA
